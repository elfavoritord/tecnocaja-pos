🎯 PRÓXIMOS PASOS (Instrucciones para usar)
═══════════════════════════════════════════════════════════════════════════

PASO 1: CONFIGURAR FIREBASE (5 MINUTOS)
────────────────────────────────────────

1. Abre: https://console.firebase.google.com
2. Selecciona tu proyecto: "inversiones-martinez-14703"
3. Ve a: Firestore Database → Rules
4. Copia TODO el contenido de FIRESTORE_RULES.txt
5. Pega en el editor y haz clic en "Publish"
6. Descarga credenciales de Admin:
   - Project Settings → Service Accounts → Generate Private Key
   - Guarda el JSON descargado en lugar seguro
7. Establece variable de entorno:
   export FIREBASE_KEY_PATH=/ruta/al/archivo.json

PASO 2: INICIAR TECNO_CAJA (2 MINUTOS)
────────────────────────────────────

1. Abre terminal en la carpeta del proyecto
2. Ejecuta: npm run desktop (o npm start)
3. Tecno Caja inicia automáticamente con sincronización activada

PASO 3: MONITOREAR SINCRONIZACIÓN (1 MINUTO)
───────────────────────────────────────────

1. Abre navegador
2. Ve a: http://localhost:3399/html/firebase-config.html
3. Verás estado en tiempo real

PASO 4: CREAR VENTA DE PRUEBA (1 MINUTO)
──────────────────────────────────────

1. En Tecno Caja: Completa una venta normal
2. Márcala como "Pagada"
3. En firebase-config.html: Verifica que aparezca en cola
4. Espera 30 segundos: Debería cambiar a "Sincronizado"

PASO 5: ABRIR APP FLUTTER (1 MINUTO)
───────────────────────────────────

1. Abre otra terminal
2. Navega a: cd "reporte app"
3. Ejecuta: flutter run (o flutter run -d chrome para web)
4. Inicia sesión con usuario Firebase Auth
5. Abre pantalla de reportes
6. ¡Verás la venta sincronizada! ✓
