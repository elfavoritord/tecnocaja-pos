¿Cómo trabajará este sistema?

Elige la estructura de operación. Esto define cómo se conectan las cajas, sucursales y el panel central.

Monocaja: una caja independiente. Multicaja: varias cajas en la misma red. Sucursal: terminal secundaria vinculada al sistema principal. Multisucursal: múltiples locales con panel central.

Autenticación obligatoria

Para continuar con esta configuración, inicia sesión con una cuenta de administrador creada previamente en el sistema principal.

Modo: Multicaja
Escribe la Dirección IPv4 real de la PC principal o usa el botón de búsqueda automática. Puede ser 192.168.x.x, 10.x.x.x o 172.16-31.x.x. No uses la puerta de enlace del router.
Cómo encontrar la IP correcta
1. En la PC principal abre CMD.
2. Escribe ipconfig.
3. Busca Dirección IPv4.
4. Usa esa IP, no la Puerta de enlace predeterminada.
Correcto: 192.168.1.25:3399, 10.0.0.25:3399 o 172.16.0.25:3399
Incorrecto: 192.168.1.1:3399 o 10.0.0.1:3399 si esa IP es el router.
Si falla, en la PC principal permite Node.js en el Firewall de Windows o abre el puerto TCP 3399.
netsh advfirewall firewall add rule name="Tecno Caja Puerto 3399" dir=in action=allow protocol=TCP localport=3399

Monocaja usa una sola sucursal con una sola caja. Multicaja usa una sucursal con varias cajas. Multisucursal habilita varias sucursales con sus cajas.

Sucursales y cajas

Las aperturas, cierres, ingresos, egresos y ventas nuevas se registrarán en la sucursal y caja activas.
Caja obligatoria para cajeros
Cuando esté activa, el formulario Nuevo Usuario exigirá una caja para roles de cajero en multicaja y multisucursal.
Asignación exclusiva por caja
Evita crear dos cajeros activos con la misma caja fija.
Define el rol de cada caja: Mixta cobra y factura, Facturación solo emite facturas (pasan a cola de cobro), Cobro solo cobra facturas pendientes.

⚠️ Eliminar sucursales y cajas

Solo visible para el administrador general. No puedes eliminar la sucursal o caja activa, ni la última de cada tipo.

Gaveta Registradora

Abrir gaveta automáticamente al cobrar
La gaveta se abre al finalizar cada venta cobrada.
La mayoría de gavetas se conectan al puerto RJ-11 de la impresora térmica.
Deja vacío para usar la misma impresora configurada para recibos.
Ejemplo: COM1, COM2, COM3...

Báscula Digital

El modo USB recibe el peso como si fuera teclado. El modo serial lee directo desde el puerto COM.
Captura automática al detectar peso
Si está activa, la pantalla de peso intentará leer la báscula automáticamente cuando abras un producto por peso.
Ejemplo: COM3, COM4 o el puerto detectado automáticamente.
Regex opcional para extraer el peso desde la cadena enviada por la báscula.
